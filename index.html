RAG Pipeline on OpenShift with Langflow

A visual Retrieval-Augmented Generation pipeline built on Red Hat OpenShift using Langflow, Docling, Milvus, and LlamaStack.

Stack

  • Langflow — Visual workflow builder (Helm chart)
  • Docling Serve — PDF extraction to Markdown (YAML manifest)
  • Milvus — Vector database, standalone mode (Helm chart)
  • LlamaStack — Granite embedding-125m + Qwen3-0.6B LLM (OpenShift AI)
